## 📌 Project Overview
FileForge is a Python-based file management system converted from a CLI application into a web application using Streamlit. It enables users to perform core file operations such as creating, reading, updating, and deleting files directly from a browser.

---

## ✨ Features
- 📄 Create new files with custom content  
- 📖 Read and display file content  
- ✏️ Update files (rename, overwrite, append)  
- 🗑️ Delete files with confirmation  
- 📂 View and select files dynamically  
- ⚠️ Error handling for invalid operations  

---

## ⚠️ Challenges Faced
### 1. CLI to Web Conversion  
Replacing `input()` and `print()` with Streamlit components.

### 2. File Path Issues  
Handling incorrect paths and missing files.

### 3. Environment Setup Problems  
Streamlit and pip not recognized in terminal.

---

## 🛠️ Solutions Implemented
- Used `st.text_input`, `st.selectbox`, `st.text_area` for UI  
- Used `Path.cwd()` for reliable file access  
- Used `python -m streamlit` to resolve command issues
