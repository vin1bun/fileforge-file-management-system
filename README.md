# fileforge-file-management-system
A command-line file management application built using Python. FileForge allows users to perform essential file system operations in a structured and user-friendly way.
• Create files  
• Read file content  
• Update (rename, overwrite, append)  
• Delete files  
• Recursively list files and folders  

Built using pathlib for modern filesystem handling
and structured error handling for reliability.
